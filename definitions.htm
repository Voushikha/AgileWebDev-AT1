Definitions

Programming Concepts

Authentication is the process by which a user, service, or app proves its identity to gain access to organizational resources, typically involving usernames, passwords, or other forms of identification like biometrics or certificates issued by trusted third parties. Web Security protects networks, servers and computer systems from any infraction to the system.

Hypertext Transfer Protocol (HTTP) is a foundational protocol for exchanging resources like HTML documents on the web, between a client and a web server.

Session management is the process used to securely handle multiple requests between the user and the web application.

Stateless programming means that the program that does not store data or information about the client session.

Types of Technologiess

• Sequence refers to the order in which instructions occur and are processed in a program. • Selection determines the path a program takes during execution based on certain conditions. • Iteration involves the repeated execution of a specific section of code while the program is running.

Hypertext markup language (HTML) defines the meaning and structure of the web content and is to be displayed on a web browser. Markup languages are used to define the text document within the tag which defines the structure of web pages and also used to store and exchange data.

The Cascading style sheets (CSS). is a stylesheet language to display the layout and structure of web pages like HTML and XML.

Syntax in programming languages refers to the set of rules that dictate how statements and expressions are correctly structured using symbols, keywords, and punctuation. It defines the structure of the language, allowing programmers to communicate effectively with the computer through precise instructions.
